# STEAM Teacher Certification Platform (Frontend)

Frontend-only scaffold for a paid teacher training & certification platform:
- STEAM ONE / TWO / THREE programs (paid)
- Checkout + payment placeholder flow
- Dashboards (Teacher / Facilitator / Admin)
- Certified Teachers Marketplace with admin-mediated contact request
- Recruit-a-Teacher form
- Consulting services pages
- Nigeria time (Africa/Lagos) shown in the sidebar
- Microsoft Education logo placeholder always visible (replace SVG in /public)

## Tech
- React + TypeScript
- Plain CSS (no Tailwind, no UI libraries)

## Run
```bash
npm install
npm run dev
```

## Replace Logo
Replace `/public/microsoft-education-placeholder.svg` with your official brand asset.

## Backend Hook Points
See `src/services/api.ts` for a placeholder API layer.
